
# Real-Time Mobile Detection in Restricted Areas

A computer vision-based surveillance system developed using Jetson Nano for detecting unauthorized mobile phone usage in restricted zones.

## Technologies Used
- Python
- OpenCV
- Computer Vision
- Jetson Nano
- Deep Learning

## Dataset
- COCO Dataset

## Features
- Real-time detection
- Edge AI deployment
- Mobile phone identification
- Low-latency inference
